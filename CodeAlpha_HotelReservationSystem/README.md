# CodeAlpha_StudentGradeTracker

**Java Programming Internship Task 1 @ CodeAlpha**

**Project: Student Grade Tracker**

Console-based Java application to manage student grades.

**Features:**
- Input multiple students and their grades for different subjects
- Calculate average, highest, and lowest scores per student
- Display a clear summary report
- Basic input validation for grades (0-100 range)

**Technologies Used:**
- Java (OOP concepts: Classes, ArrayList, Scanner)
- No external libraries

**How to Run:**
1. Compile: `javac StudentGradeTracker.java`
2. Run: `java StudentGradeTracker`

**Example Usage:**
- Enter number of students
- Enter number of subjects
- Input names and grades (numbers only)
- See the summary report

Submitted as part of CodeAlpha Java Internship.
