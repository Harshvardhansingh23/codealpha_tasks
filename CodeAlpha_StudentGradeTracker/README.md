# CodeAlpha_HotelReservationSystem

**Java Programming Internship Task 4 @ CodeAlpha**

**Project: Hotel Reservation System**

Console-based hotel booking management system.

**Features:**
- View available rooms by category (Standard, Deluxe, Suite) or all
- Book a room with guest name, room number, and date
- Cancel existing booking by room number
- View all current bookings
- Simulated payment on booking
- Input validation for numbers

**Technologies Used:**
- Java (OOP: Classes for Room & Reservation, ArrayList for data storage)
- Scanner for user input

**How to Run:**
1. Compile: `javac HotelReservationSystem.java`
2. Run: `java HotelReservationSystem`

**Sample Rooms (pre-added):**
- 101, 102 → Standard (~₹2500)
- 201, 202 → Deluxe (~₹4500)
- 301 → Suite (~₹8500)
